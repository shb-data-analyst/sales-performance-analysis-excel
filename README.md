# sales-performance-analysis-excel
Sales performance analysis using Microsoft Excel(Pivot Tables and charts)

## Project Overview
This project analyzes retail sales data to identify sales trends and
regional performance using Microsoft Excel.

## Dataset
Sample retail sales dataset (practice data)

## Tools Used
Microsoft Excel

## Techniques Applied
- Pivot Tables
- Pivot Charts
- Date Grouping (Year and Month)
- Trend Analysis

## Key Insights
- Sales increased from 2023 to 2024.
- Sales in 2025 are lower due to partial-year data.
- North region contributes the highest total revenue.

## Project Files
- Sales by Region analysis
- Sales Trend analysis
- Insights summary

## Outcome
This project demonstrates practical Excel data analysis skills
commonly required in business and freelance data analysis tasks.
